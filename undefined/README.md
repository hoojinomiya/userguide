사용자 로그인
=============

Cloud Z사용자 포탈을 접속하여 오른쪽 상단 로그인 버튼을 클릭하면 다음과 같이
아이디와 비밀번호를 입력하는 페이지가 나타납니다. 올바른 계정 정보를 입력 후
로그인 시 Cloud Z 포털 메인화면으로 이동되며, 계정정보가 기억나지 않는다면
\<아이디 찾기\>와 \<비밀번호 찾기\>를 통해 관련 정보를 찾을 수 있습니다.

![](media/5869b195a2c68712eb90d518a491acd7.png)

-   로그인 화면  
    

    ![](media/2e8834c4b8d41c0ed584360bc97e6206.png)

만약 비밀번호를 잘못 입력하면 다음과 같은 화면이 나타납니다.

![](media/97b37b86162f9007b35dcbd2faf32bb8.png)

비밀번호를 5번 이상 잘못 입력하면 해당 계정이 잠김 처리됩니다. 재 로그인은 30분
이후 가능하며, 바로 잠김 해제를 원하시면 Tech Center(02-6400-2222)로 문의
바랍니다.

![](media/01fe17fc980cf458e6fa0477f66b5289.png)

회원가입
========

메인화면 상단의 \<로그인\> 버튼을 클릭하면 화면 하단의 \<회원가입\> 버튼을
확인하실 수 있습니다.

**회원정보 입력**
-----------------

### 회원타입 선택

\<회원가입\> 버튼을 선택하면 다음과 같이 어떤 타입의 회원으로 가입하고 싶은지
묻는 화면으로 이동합니다. 원하는 타입을 선택 후 \<다음\> 버튼을 선택합니다. 개인
사용자의 경우 무료체험 가입을 통해 이용 가능합니다.

![](media/718b2978519233494190c4f5ec511a30.png)

### 약관동의

\<다음\> 버튼을 선택하면 다음과 같이 선택한 회원타입 ‘약관동의’ 화면으로
이동합니다. 전체 약관에 동의하면 \<다음\> 버튼이 활성화됩니다. 각각의
약관동의들을 선택하거나, [클라우드 제트 전체 약관에 동의 합니다]를 선택하여 한
번에 모든 약관에 동의할 수 있습니다.

![](media/998a37c8507b520a1563a0f4bc771b44.png)

### 회원정보 입력

각각의 항목에 해당하는 정보를 입력 후 \<가입 신청하기\> 버튼을 선택합니다. Cloud
Z 관련 정보를 E-mail로 수신 받고자 한다면 아래에 있는 ‘*E-mail 수신에
동의합니다*’를 선택합니다.

![](media/271acf9144db74fad0e614b0823d7695.png)

### 가입완료

\<가입 신청하기\> 버튼을 선택하면 다음과 같이 [가입완료] 화면이 나타납니다.

![](media/be17a0e3422bf56f1231ddd581c1a41b.png)

| Note 기업회원과 파트너회원은 회원가입 후 즉시 사용자 포털에 로그인은 불가능하며, 관리자의 가입승인 필요합니다. |
|----------------------------------------------------------------------------------------------------------------|


아이디와 비밀번호 찾기
----------------------

로그인 화면에서 \<아이디 찾기\> 버튼 또는 \<비밀번호 찾기\> 버튼을 선택합니다.

### 아이디 찾기

\<아이디 찾기\> 버튼을 선택하면, 다음과 같이 [아이디 찾기] 페이지로 연결됩니다.
가입할때 입력했던 이메일 주소를 입력 후 \<아이디 찾기\> 버튼을 선택합니다.

![](media/79bc5c2188fcaf3e397e49115b3ebe77.png)

입력한 email 주소가 유효하다면 [아이디 찾기] 완료 페이지로 이동되며, 입력한 메일
주소로 안내메일이 발송됩니다. \<확인\> 버튼을 선택하면 Cloud Z 로그인 페이지로
이동합니다.

![](media/dc162fa9d7f19dc331b2ab913e9d462a.png)

### 비밀번호 찾기

\<비밀번호 찾기\> 버튼을 클릭하면, 다음과 같이 [비밀번호 찾기] 페이지로
이동합니다. 가입 시 입력했던 이메일 주소와 아이디를 각각 입력 후 \<비밀번호
찾기\> 버튼을 선택합니다.

![](media/401a0ab4556e1cb1b9573819148440ef.png)

입력한 정보가 유효하다면 비밀번호 찾기 완료 페이지가 나타납니다. **메일로 전달된
Link를 통해 비밀번호를 재설정 후 서비스를 이용할 수 있습니다**. \<확인\> 버튼을
선택하면 Cloud Z 로그인 페이지로 이동합니다.

![](media/5e798b66d9c8041ae3dcc7dec73bae39.png)

제품 확인 및 주문/관리 
=======================

제품 확인
---------

### [홈포털] 제품 소개

홈포털 메인화면 또는 상단 메뉴의 제품을 클릭하면 Cloud Z에서 제공하는 제품들을
만나보실 수 있습니다.

![](media/236aeb106a2f9e8b3100c3912427995d.png)

### [홈포털] 요금계산기

요금계산기에서는 Cloud Z에서 주문가능한 상품 및 옵션에 따른 가격을 쉽게 확인할
수 있습니다. 홈포털 화면의 오른쪽 Quick 메뉴에서 요금계산기 아이콘을 클릭해
주세요.

요금계산기의 상단에서 먼저 제품군 및 제품명을 선택 후 차례대로 기본선택 (데이터,
요금제, 타입)과 상품 및 옵션을 선택하면 아래 이미지와 같이 월별 예상요금이
산정됩니다. 선택된 관심있는 상품들은 구성리스트에 담을 수 있으며, 하단의 엑셀
다운로드 버튼을 누르면 구성리스트의 내역이 담긴 파일을 다운로드할 수 있습니다.

![](media/a0709cd7abb2481d7b38b2b6f758ca2a.png)

| Note 요금계산기의 결과는 추정 요금으로 실 사용 요금과 다를 수 있습니다. |
|-------------------------------------------------------------------------|


### [콘솔] 제품 전체보기

[콘솔] 화면 내 왼쪽 상단의 \<제품 전체보기\> 버튼을 클락하면 IBM Cloud, Alibaba
Cloud 및 Cloud Z의 현재 주문 가능한 제품 목록을 확인할 수 있습니다. 각 제품을
클릭하면 해당 제품을 주문할 수 있는 페이지로 이동합니다.

![](media/b833660b5330655b7c1ac32cb9558e4d.png)

제품 주문
---------

제품 제품 주문 및 관리는 모두 [콘솔]에서 이루어집니다. 클라우드 포털화면 상단의
\<콘솔\> 버튼 클릭 시 콘솔 페이지로 이동합니다.

![](media/e44fd206fdac8b50a5ca8fe8523c0681.png)

제품 중 하나를 클릭하면 아래 이미지와 같이 해당 제품의 첫번째 타입의 상품들이
보여지며, 다른 타입의 상품을 확인 시 상단의

![](media/e34943cc9d71dbcfa17d8f1405faf2db.png)

빨간 버튼을 클릭합니다. 목록 상단에서 제품에 따라 요금제 타입 및 데이터 센터를
선택할 수 있으며, 센터별로 가격이 상이할 수 있습니다.

![](media/19394ef04d2b56e9264f19ba4694b682.png)

상품명의

![](media/50be06d5ed4f770e478c7ace1da9658e.png)

아이콘에 마우스를 갖다대면 해당 상품의 포함스펙을 확인할 수 있습니다.

![](media/97cbb8a3c5650e10e9127981cb9b16c0.png)

상품 정보 조회 및 설정 변경
---------------------------

[전체] 탭 화면에서 상품의 데이터센터, 상품명, 시작가격 및 기본 Spec정보 및 화면
하단에서 해당 상품의 자세한 설명을 확인할 수 있습니다.

![](media/52e587085b7bd0ea9b57306484cfcc07.png)

[상품 구성] 탭에서 \<수정\> 버튼을 선택하면 해당 항목들의 설정들을 변경할 수
있으며, \<옵션 더보기\> 버튼을 선택하여 추가 항목들을 확인 및 변경할 수
있습니다.

![](media/8a94b8651b93107c05ac80bccf3dfa1b.png)

상품 주문
---------

### 주문하기

원하는 항목들의 설정을 끝낸 뒤 주문페이지 하단에 월 비용 확인 후 \<주문하기\>
버튼을 선택하여 해당 상품을 주문합니다.

![](media/40aee9955af71144672c978c5aed1ea6.png)

### 주문 확인 팝업

\<주문하기\> 버튼을 선택하면 다음과 같이 주문 정보를 요약한 [주문 확인] 팝업이
나타납니다. 주문 상품의 금액 확인 및 주의 사항에 체크 후 \<확인\> 버튼을
선택하여 주문을 완료합니다.

### [./media/image24.png](./media/image24.png)

주문 완료 팝업

주문이 정상적으로 완료 시 다음과 같은 팝업이 나타납니다.

![](media/15ccbeb5982739276aa37071077f9fdf.png)

주문 완료 팝업에서 \<확인\>를 선택하면 상품 선택 화면으로 이동합니다.

상품 관리
---------

### 주문한 상품 확인 및 관리

왼쪽메뉴의 해당 제품명을 클릭하면 주문한 상품을 목록에서 확인할 수 있으며,

서버명 또는 조치 버튼을 클릭 시 해당 제품의 상세 정보 확인 및 기능을 사용할 수
있습니다.

![](media/36f09bb1db8a9e32e91497fc82e27bb1.png)

My Page
=======

\<My Page\>는 로그인 후 이용가능하며 회원정보 관리를 제외한 이용현황, 청구서
조회, 주문 조회, 온라인 상담/문의 조회, 작업공지, Ticket 관리 기능은 [콘솔]에서
이용가능합니다.

이용현황 Dashboard 
-------------------

공지사항 및 이용하고 있는 서비스 청구 금액 확인이 가능하며 주문내역, 최근 문의
및 Ticket 내역 등의 다양한 정보를 확인할 수 있습니다.

![](media/0ad4b3fbb4d830a650612957c8368454.png)

청구서 조회
-----------

화면 상단의 사용자명 \> 청구서 조회를 클릭하면 [콘솔]의 해당 화면으로
이동합니다. 공급사별 월별 청구 내역 및 청구서 상세 내역을 확인할 수 있으며 해당
월의 청구서 내역을 다운로드할 수 있습니다. [최근 6개월 청구 금액] 탭을 클릭하면
최근 6개월간의 평균 청구 금액을 그래프로 확인 가능합니다.

![](media/76fdac95c9f616ccfadd9103a0b73760.png)

![](media/34b355d3023a04b5233ff212f3c8a612.png)

주문 조회
---------

화면 상단의 **사용자명 \> 주문조회**를 클릭하여 [콘솔]의 해당 화면으로
이동합니다. 주문한 상품에 대한 조회가 가능하며, 상단의 검색바를 통하여 기간,
상태, 주문타입을 선택하여 조회가 가능합니다.

![](media/4f081e39947a0a23f5eda1021715ada6.png)

온라인 상담/문의 조회
---------------------

화면 상단의 **사용자명 \> 온라인 상담/문의 조회**를 클릭 또는 [홈포털] 메뉴의
고객지원 \> 온라인 상담/문의를 클락하면 해당 화면으로 이동합니다. 로그인한
회원은 [콘솔]에서, 비회원은 [홈포털] 화면에서 상담/문의 내역 조회 및 관리가
가능합니다.

![](media/b59516859374470dfd717cdbe8d107d1.png)

회원정보관리
------------

[홈포털] 또는 [콘솔] 화면 상단의 **사용자명 \> 회원정보관리**를 클릭하면
회원정보 확인 및 수정 화면으로 이동합니다. ‘회원정보 수정’, ‘비밀번호 변경’,
‘결제정보 관리’, ‘회원 탈퇴’ 기능을 이용할수 있으며, 해당화면은 로그인한 계정의
비밀번호를 입력 후 진행이 가능합니다.

### 회원정보 수정

회원정보 수정은 휴대폰과 E-mail만 가능합니다.

![](media/ca13394c5cecb62f67bbb89e406f60c1.png)

### 비밀번호 변경

현재 비밀번호를 입력 후 변경할 비밀번호를 입력하여 변경합니다.

![](media/0a2d19b2faafa649df320c1e8250e550.png)

### 결제정보 관리 (개인 회원만 해당)

현재 사용중인 카드 정보 확인 및 변경 기능을 제공합니다. 카드 정보를 변경하려면
\<수정\> 버튼을 선택 후 팝업의 안내를 따라 변경할 카드 정보를 입력하여
변경합니다.

| Note 입력된 카드 정보는 일체 Cloud Z 시스템에 저장되지 않으며 식별을 위한 카드사 및 고유 식별 번호만 저장합니다. |
|------------------------------------------------------------------------------------------------------------------|


![](media/300315ded4b611120e3ed673d0fcd79d.png)

### 회원 탈퇴

회원 탈퇴 시 제공되는 모든 주의사항을 확인 후 ‘회원 탈퇴에 관한 모든 사항을
확인하였으며, 이에 동의합니다.’에 동의 후 \<탈퇴 신청\> 버튼을 선택하여
진행합니다.

| Note 회원 탈퇴는 현재 계정에 청구 금액이 없을 경우에만 가능합니다. 미납금이 존재하는 경우 납부 후 탈퇴 처리가 가능합니다. |
|---------------------------------------------------------------------------------------------------------------------------|


![](media/7bb887dde3e50948fe03bc2b4bfc828f.png)

### Ticket 조회

화면 상단의 **사용자명 \> Ticket관리**를 클릭하여 [콘솔]의 해당 화면으로
이동합니다. 작성한 Ticket 조회 및 상태를 확인할 수 있으며, update 일시를
확인하여 최근 업데이트 여부를 확인합니다.

![](media/f1adca0db0d59bc9951ee1fec8155314.png)

상태가 Open인 Ticket은 추가적인 Update로 진행 중 추가 문의가 가능합니다.

(Close 된 Ticket은 추가적인 Update가 불가능합니다.)

![](media/06bcc146a108bc70cd662ad34d4365ce.png)

### Ticket 등록

Ticket관리 화면에서 \<+Ticket 발행\> 버튼을 클릭하면 Ticket 발행을 위한 팝업창이
노출됩니다. 작성하려는 내용에 맞는 주제(API, Network, Storage 등), 제목, 내용
등을 입력 후 Ticket update 알림을 받기 위한 E-mail을 입력합니다.

-   여러명이 Update 알림을 받아야한다면 ‘;’ 구분자를 넣어 다수의 E-mail을
    입력합니다.

-   현재 Ticket과 관련된 장비가 있다면, Device를 선택 후 Ticket 발행합니다.

-   Ticket 등록은 Global Cloud 사업자에게 직접 문의하는 것으로 한글 Ticket 은
    지원이 불가합니다. 영문으로 작성해 주세요.

![](media/e6fc3e46e869dca2b18a79efb5b4431a.png)

### 본인인증 

개인회원은 [휴대폰 인증] 또는 [아이핀 인증]을 통한 본인 인증을 진행합니다.

![](media/2ff579eca6242eaa9e7367f1b107df1e.png)

-   **휴대폰 인증**

>   [휴대폰 인증]을 이용 시 \<인증하기\> 버튼을 선택하면 다음과 같이 이용중인
>   통신사를 선택하여 본인 인증을 진행합니다.

![](media/bb61e3ff499b85385f4fdc8535e0a13d.png)

-   **아이핀 인증**

>   [아이핀 인증] 이용 시 \<인증하기\> 버튼을 선택하면 다음과 같이 인증을 위한
>   창이 뜨며 ‘아이핀ID’, ‘비밀번호’, ‘문자입력’을 입력하여 1차 인증을 진행하며,
>   2차 비밀번호 입력 후 \<확인\> 버튼을 선택하면 본인 인증 처리가 완료됩니다.

![](media/3eb95bb6e79d95b0d6def1825b1f8653.png)
